# Caça-Palavras
O produto proposto se trata de um jogo de caça palavras feito com a linguagem python. 
o objetivo do produto é estabelecer ao jogador dicas relacionadas as palvras que estão na estrutura(caça-palvras)
como támbém uma função que permita ao jogador mudar de caça palavra caso ele deseje. 
a ideia é simple, trazer ao jogo clássico de infância uma nova estrutura dessa vez em forma tecnólogica é dinâmica.
